# Веб-приложение с микросервисной архитектурой (Dota 2 Heroes Biography)

Веб-приложение для просмотра биографий персонажей Dota 2 с микросервисной архитектурой.

## Быстрый старт

**Подробная инструкция:** [КАК_ЗАПУСТИТЬ.md](./КАК_ЗАПУСТИТЬ.md)

1. Установите Docker Desktop
2. Откройте PowerShell и выполните:

```powershell
cd infra
docker compose up --build
```

3. Откройте http://localhost:3000

## Создание администратора

```bash
docker exec -it dota2-postgres psql -U postgres -d auth_db
UPDATE users SET role = 'admin' WHERE email = 'ваш_email@example.com';
```

## Добавление биографий

1. Зарегистрируйтесь и войдите как администратор
2. Перейдите в админ-панель
3. Выберите персонажа и добавьте биографию

## Структура

- `frontend/` - React приложение
- `services/` - Микросервисы (auth, heroes, lore)
- `infra/` - Docker Compose

## Архитектура

Приложение состоит из 4 основных компонентов:

1. **Frontend (React)** - пользовательский интерфейс
2. **Auth Service** - сервис аутентификации (JWT)
3. **Heroes Service** - сервис управления персонажами
4. **Lore Service** - сервис связей и избранного

Все сервисы взаимодействуют через REST API и используют PostgreSQL для хранения данных.

## Персонажи

В базе данных уже добавлено **124 персонажа** Dota 2 (все основные герои). 
Биографии можно добавить через админ-панель.
